DKARL TUNNEL Privacy Policy

Effective date: May 23, 2026

DKARL TUNNEL is a VPN and tunnel configuration app for advanced users. This page explains what information the app may process, how ads and consent work, and how users can manage privacy choices.

Information We Process

The app may process configuration values entered by the user, including server addresses, ports, payloads, SNI/SSL values, DNS settings, proxy settings, tunnel mode choices, and imported .dkl configuration files. These values are used to run the VPN/tunnel features selected by the user.

DKARL TUNNEL may also read basic device and network state needed for VPN operation, such as whether a network is connected, whether mobile data is active, and carrier/operator information when a config is locked to a mobile network or operator.

VPN And Network Traffic

When the VPN is started, traffic may be routed through the server or tunnel configuration chosen by the user. DKARL TUNNEL does not provide the remote server itself; the privacy and logging behavior of any remote server depends on the server owner and configuration used.

Advertising And Consent

DKARL TUNNEL uses Google AdMob to display ads. Google and its partners may collect identifiers, device information, and ad interaction data to provide, measure, and improve advertising, subject to the consent choices shown in the app.

For users in regions where consent is required, DKARL TUNNEL uses Google's User Messaging Platform consent form before requesting ads. Ads are not requested until the Google consent status allows ad requests.

Manage Or Revoke Consent

Users can manage or revoke advertising consent inside the app by opening DKARL TUNNEL, going to the menu, and choosing Privacy options when it is shown. If the option is not shown, Google has not required a privacy options entry point for that user or region at that time.

This page can be used as the privacy and revocation information link for AdMob: https://dkarl.site/privacy/#revoke-consent.

Configuration Files

Imported or exported configuration files may contain server credentials and tunnel settings. Users should only import files from trusted sources. Protected configs may restrict editing or exporting inside the app, but no offline app file can be guaranteed impossible to reverse engineer.

Third-Party Services

The app may use Google AdMob, Google Play services, Google Play review APIs, and other Android libraries needed for app features, ads, stability, and user experience. These services may process data under their own privacy policies.

Security

DKARL TUNNEL uses reasonable safeguards for local configuration handling, but users should avoid sharing sensitive configs and should only use trusted servers. VPN and tunnel security depends on the configuration and remote infrastructure selected by the user.

Children

DKARL TUNNEL is intended for advanced users and is not directed to children. Users who are not old enough to consent to data processing in their region should not use the app without a parent or guardian.

Changes

This policy may be updated when the app, advertising setup, or legal requirements change. The effective date at the top of this page will be updated when material changes are made.

Contact

For privacy questions about DKARL TUNNEL, contact the developer through the support contact listed on the app's Google Play Store page.
